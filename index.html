어떤 분들을 위한 과정인가요?

한의·한방 산업 재직자

"환자 데이터는 쌓여가는데, 어떻게 활용해야 할지 모르겠어요."

  • EMR 데이터로 맞춤형 진료 서비스를 제공하고 싶은 분
  • 데이터 기반으로 한의원·한방병원 경쟁력을 높이고 싶은 분
  • 한방 제품 소비자 패턴을 분석해 시장 기회를 찾고 싶은 분

식품 산업 재직자

"빠르게 변하는 소비자 트렌드, 데이터로 예측하고 싶어요."

  • 판매/고객 데이터를 분석해 히트 상품 개발의 실마리를 찾고 싶은 분
  • K-푸드 세계화 전략에 맞춰 데이터 기반 해외 진출 전략을 세우고 싶은 분
  • 푸드테크·기능성 식품 등 신사업 성공 가능성을 높이고 싶은 분

뷰티 산업 재직자

"감(感)에 의존하는 마케팅은 이제 그만! 숫자로 이야기하고 싶어요."

  • 리뷰·판매 데이터를 분석해 소비자 심리를 파악하고 싶은 분
  • K-뷰티 트렌드를 예측하고 데이터 기반 신제품 전략을 세우고 싶은 분
  • 온라인 플랫폼 판매 성과를 분석하고 매출을 극대화하고 싶은 분

무엇을 배우게 되나요?

1. 데이터 분석 기초 다지기 (온라인)

SQL 기초 – 데이터 다루기 첫걸음

SQL의 핵심 문법을 통해 데이터를 선택, 필터링, 입력하는 방법을 익힙니다.

SQL 실전 – 데이터를 정리하고 연결하기

GROUP BY, JOIN, 서브쿼리를 활용해 다중 테이블 데이터를 정리하고 분석합니다.

Python & Pandas – 복잡한 데이터 쉽게 다루기

Pandas를 이용해 다양한 데이터 형식을 정리하고, 데이터 처리 실습을 진행합니다.

시각화 & 생성형 AI 마케팅 실습

시각화 도구와 생성형 AI(GPT 등)를 활용하여 마케팅 데이터를 분석하고 자동화합니다.

2. AI 마케팅 실무 역량 강화 (오프라인)

1

산업 데이터 이해 및 정리 실습

산업별 주요 데이터를 분석 가능한 형태로 정제하고, 핵심 인사이트를 도출합니다.

2

마케팅 데이터 분석 실습

소비자 행동, 매출 등의 데이터를 활용한 분석과 시각화 실습을 진행합니다.

3

AI 분석 자동화 & 해커톤

생성형 AI를 활용해 분석과 시각화 자동화를 실습하고 해커톤으로 문제 해결 능력을 강화합니다.

4

마케팅 전략 수립 및 캡스톤

데이터 기반 마케팅 전략 기획서 작성과 팀 프로젝트 발표로 실무 역량을 완성합니다.

왜 이 과정을 선택해야 할까요?

🎯

산업 특화 커리큘럼

한의, 식품, 뷰티 산업 데이터를 활용하여 현업 적용 가능성을 극대화했습니다.

🛠️

실무 중심 프로젝트(PBL)

산업 현장 데이터로 해커톤, 캡스톤을 진행하며 실전 문제 해결 능력을 강화합니다.

✅

검증된 교육 전문성

대구한의대 K-MEDI 노하우와 유다시티 학습 모델이 결합된 고품질 교육을 제공합니다.

🔄

온·오프라인 융합 교육

온라인에서 기초를, 오프라인 캠프로 실무 역량을 완성하는 효율적 학습을 경험하세요.

🏆

공신력 있는 인증 및 지원

수료 시 디지털 배지를 발급하고, 우수 수료생을 위한 경력 개발을 지원합니다.
